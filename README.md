# Curso TeoMeWhy Git e Github 2026  

Um curso voltado para iniciantes aprenderem a trabalhar com versionamento de código e repositórios remotos com Github.

Aprenda na prática o sistema de controle de versão mais usado do mercado. Do primeiro commit até a colaboração em equipe com pull requests, branches e resolução de conflitos. Ideal para quem está começando ou deseja consolidar boas práticas no versionamento de código. Inclui autenticação moderna (tokens e SSH), exercícios guiados e um projeto real. Ao final, você terá um portfólio no GitHub e estará pronto para trabalhar em qualquer equipe de desenvolvimento.

Esse curso está sendo realizado pelo Youtuber/Estatístico Teo o Calvo, seu canal e site está logo abaixo:

- [Curso Git 2026](https://youtube.com/@teomewhy)
- [teomewhy.org](https://teomewhy.org/schedule)
